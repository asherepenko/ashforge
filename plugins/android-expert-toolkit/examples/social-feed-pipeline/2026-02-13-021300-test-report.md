## Test Doubles Created

### TestFeedRepository

**Location:** `core/testing/src/main/kotlin/com/example/core/testing/data/TestFeedRepository.kt`

**Purpose:** Controllable FeedRepository for ViewModel and UI testing

**Implementation:**
```kotlin
class TestFeedRepository : FeedRepository {

    private val _posts = MutableStateFlow<PagingData<Post>>(PagingData.empty())
    override val posts: Flow<PagingData<Post>> = _posts.asStateFlow()

    private val likedPosts = mutableSetOf<String>()
    private var shouldFailLike = false

    /**
     * Emit posts for testing.
     */
    fun emitPosts(posts: List<Post>) {
        _posts.value = PagingData.from(posts)
    }

    /**
     * Simulate like failure for error testing.
     */
    fun setShouldFailLike(shouldFail: Boolean) {
        shouldFailLike = shouldFail
    }

    override suspend fun likePost(postId: String) {
        if (shouldFailLike) {
            throw IOException("Network error")
        }
        likedPosts.add(postId)
    }

    override suspend fun refresh() {
        // No-op in test - refresh is simulated by emitting new posts
    }

    fun isPostLiked(postId: String): Boolean = postId in likedPosts
}
```

**Usage in Tests:**
```kotlin
@Test
fun `likePost updates UI state correctly`() = runTest {
    val testRepo = TestFeedRepository()
    val viewModel = FeedViewModel(testRepo)

    // Emit test posts
    testRepo.emitPosts(listOf(testPost1, testPost2))

    // Like a post
    viewModel.likePost("post-1")

    // Verify repository called
    assertTrue(testRepo.isPostLiked("post-1"))
}
```
